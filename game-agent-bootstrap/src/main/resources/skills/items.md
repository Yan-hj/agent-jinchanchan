# 金铲铲之战 — 装备合成与推荐

## 基础散件
- 大棒（法术强度）
- 眼泪（法力值）
- 大剑（攻击力）
- 反曲弓（攻击速度）
- 锁子甲（护甲）
- 腰带（生命值）
- 魔抗斗篷（魔法抗性）
- 拳套（暴击率/闪避）

## 法师核心装备
| 装备 | 合成 | 效果 |
|------|------|------|
| 蓝 buff | 眼泪 + 眼泪 | 法力值降至 10，释放技能后回蓝 |
| 青龙刀 | 大剑 + 眼泪 | 普攻额外回蓝 |
| 法爆 | 大棒 + 拳套 | 技能可暴击 |
| 帽子 | 大棒 + 大棒 | 法术强度 +50% |
| 科技枪 | 大棒 + 大剑 | 技能伤害治疗自身 |

## 坦克装备
| 装备 | 合成 | 效果 |
|------|------|------|
| 日炎 | 锁子甲 + 腰带 | 每秒对周围造成伤害 |
| 反甲 | 锁子甲 + 锁子甲 | 免疫暴击额外伤害 |
| 狂徒 | 反曲弓 + 腰带 | 生命值大幅提升 |
| 龙牙 | 魔抗 + 魔抗 | 魔法伤害减免 |

## 装备优先级（法师阵容）
1. **蓝 buff**：安妮/维迦核心，优先合成
2. **法爆**：维迦核心
3. **帽子**：安妮/卡牌
4. **青龙刀**：替代蓝 buff 的第二选择

## 通用装备原则
- 散件不要过早合成，等确定阵容方向
- 一件装备有 2-3 个通用备选方案
- 坦克装优先给前排核心，输出装给主 C
- 经济好时可在 4-5 后多拿装备选秀
