# 金铲铲之战 — 站位指南

## 基本站位原则
- **前排坦克**：吸收伤害，放第一排
- **后排输出**：放最后一排，远离敌方刺客
- **辅助/控制**：放在能覆盖最多友军的位置

## 法师阵容站位
- 安妮放第二排中间 → 吸收伤害不被秒，技能覆盖全场
- 维迦放第四排角落 → 最大化安全输出距离
- 控制型前排（塔姆、慎）放第一排中间
- 辅助法师放第三排两侧

## 防刺客站位
- 主 C 位缩角，旁边放一个肉盾棋子挡刺客跳位
- 或者在主 C 后面放一个棋子（敌方刺客会跳后排优先切最后排）
- 大后期可全员缩角落

## 防 AOE 站位
- 散开站位，不要密集堆叠
- 主 C 和副 C 分别放两侧，防止被一道 AOE 全秒

## 阵容对齐
- 观察敌方 C 位位置，将自己的主坦对齐敌方主 C
- 自己的 C 位避开敌方主坦和控制棋子的直线路径

## 特殊棋子站位
- **安妮**：第二排（不要第一排），保证不会第一个阵亡
- **维迦 / 卡牌**：底角，旁边放一个护卫防刺客
- **带刺转的棋子**：直接放敌方后排同一侧
